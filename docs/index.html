Manage Your Store

Everything you need to update products, categories, and settings on your MetalHub store. No technical skills required.

Getting Started

Your store is managed through a simple web interface called the CMS. When you save changes, the website updates automatically within 1-2 minutes.

Open the admin page

Go to yourdomain.com/admin in your web browser and bookmark it.

Sign in with GitHub

Use your GitHub account credentials. If you don't have access, ask your developer to set it up.

You're in

You'll see a dashboard with collections on the left: Products, Categories, Materials, Translations, and Site Settings.

Bookmark this page. You'll visit it every time you need to update your store.

Managing Products

Products are the core of your store. Each product has a name, description, images, pricing, and optional variations.

Adding a New Product

Click "Products" then "New Product"

Find Products in the left sidebar and click the "New Product" button.

Fill in the name

Enter the product name in English (required), Nepali, and Newari. The other languages can be filled in later.

Write a description

Describe the product in all three languages. Be specific about material, craftsmanship, and uses.

Upload images

Click "Add Image" to upload photos. The first image becomes the main one. Customers can click through a gallery.

Set the price

Enter the base price in Nepali Rupees (NPR). This is the starting price before any attribute changes.

Choose category and material

Select the category (e.g., Kitchenware) and material (Copper, Brass, or Bronze) from the dropdown.

Click "Publish"

Your product goes live after the site rebuilds (1-2 minutes).

Product Fields Reference

Field Required Description
Name Yes (English) Product name in all three languages
Slug Yes URL identifier (auto-generated from English name)
Description No Product description in all three languages
Category Yes Select from existing categories
Material Yes copper, brass, or bronze
Images Yes One or more product photos
Base Price (NPR) Yes Starting price in Nepali Rupees
In Stock No Toggle off to mark out of stock (default: on)
Featured No Show on homepage (default: off)

Variants (Optional)

Variants let you offer different sizes, finishes, or styles of the same product. Each variant can have its own price, discount, stock status, and images.

Example: A copper water jug in Small, Medium, and Large. Create one "Size" attribute group with three options, then three variants — each with its own price modifier.

Attribute Groups

Attribute groups define the options customers can choose:

  • Group Name — e.g., "Size", "Finish", "Color" (in all three languages)
  • Options — e.g., "Small", "Medium", "Large" (each in all three languages)

Variant Fields

Field Description
Key Unique ID (e.g., "small-polished")
Attribute Values Which group + option this variant represents
Price Modifier Added to the base price (can be negative)
Discount Optional variant-specific discount
In Stock Toggle stock per variant
Images Optional images that replace the product gallery for this variant

Discounts

Set discounts at two levels:

  • Product-level — fallback when no variant discount is active
  • Variant-level — overrides the product discount for that variant

Discounts can be percentage (e.g., 15% off) or flat (e.g., 500 NPR off). Toggle "Active" to enable.

Social Media Embeds

Each product can have Facebook, Instagram, or TikTok embeds. To add one:

  1. Go to the post on the social platform
  2. Click Share, then "Embed"
  3. Copy the embed code
  4. Paste it into the "Embed Code" field in the CMS

Managing Categories

Categories group your products (e.g., Kitchenware, Puja Items, Home Decor). Customers use these to filter products.

Adding a Category

Click "Categories" then "New Category"

Fill in the name

Enter the category name in all three languages.

Set the slug

A URL-friendly identifier (e.g., "kitchenware"). Must match exactly when assigning products.

Choose an icon

Pick an emoji to represent the category. Shows on the homepage and category pages.

Set the display order

Lower numbers appear first. Use increments of 10 (10, 20, 30) for easy reordering.

Optional: image, description, promoted

Add a banner image, description, or toggle "Promoted" to show in Hot Topics on the homepage.

Category Fields Reference

Field Required Description
Name Yes Category name in all three languages
Slug Yes URL-friendly identifier
Icon No Emoji for the category
Order No Display order (lower = first)
Image No Banner image for homepage
Description No Brief description in all three languages
Promoted No Show in "Hot Topics" on homepage

Managing Materials

Materials define what your products are made of (Copper, Brass, Bronze). Each has a name, icon, and display order.

Click "Materials" then "New Material"

Fill in the name in all three languages

Set the slug

Must match exactly: copper, brass, or bronze.

Set the icon and order

An emoji and display order (lower numbers first).

These control how customers reach you.

Important: When a customer places an order, the system builds a message with all items, prices, and a delivery map link, then sends it to this WhatsApp number or Facebook page.

Managing Translations

The site is trilingual: English (default), Nepali (/ne/), and Newari (/newa/). Most text is managed inline when editing products and categories.

The Translations collection holds UI strings — labels, buttons, and messages not tied to a specific product or category. You typically won't need to edit these.

How Publishing Works

Tip: After saving, wait 1-2 minutes and refresh your live site. Every save is a git commit — if something goes wrong, your developer can revert.

Tips & Common Tasks

1

Add a Sale Price

Open the product, scroll to "Product Discount", toggle "Active" on, choose percentage or flat, enter the value. The crossed-out price displays automatically.

2

Mark Out of Stock

Toggle off "In Stock". For products with variants, you can mark individual variants as out of stock while keeping others available.

3

Feature on Homepage

Toggle on the "Featured" switch. Featured products appear in the highlighted section on the homepage.

4

Change Images

Open the product, remove old images, upload new ones. The first image is the main one shown in product cards.

5

Add a Language

Ask your developer to add the new locale. Once added, you'll see a fourth language tab in all fields.

6

Embed Social Posts

On Facebook, Instagram, or TikTok, click Share then "Embed". Copy the code and paste it into the product's Social Media Embeds section.
